Windows Dos 시스템 및 네트워크 명령어 보기

ipconfig : 내 컴퓨터의 네트워크설정확인(IP주소확인)
ipconfig /all : 자세히표시 mac주소, dns서버등 표시
ipconfig /release : 현재 내 모든 어댑터(랜카드)의 IP(IPv4)를 반납
ipconfig /release6 : 현재 내 IP(IPv6)를 반납
ipconfig /renew : 아이피를 IP(IPv4) 다시 받아오기
ipconfig /renew6 : 아이피를 IP(IPv6) 다시 받아오기
ipconfig /displaydns : DNS 캐시 정보 보기
ipconfig /flushdns : DNS 캐시 초기화
ipconfig /registerdns : 모든 DHCP 임대를 새로 고치고 DNS 이름을 다시 등록
control netconnections : 네트워크 연결
tracert IP주소[또는 도메인] : IP 패킷이 목적지에 도달하기 위해 거치는 경로추적 Trace Route
ping IP주소[또는 도메인] : 해당IP 접속 테스트하기(접속여부, 응답시간, 패킷손실율 확인)
pathping IP주소[또는 도메인] : 경로추적과 접속테스트 같이하기(tracert + ping)
route print : 라우팅 테이블 확인
netstat : 내 컴퓨터와 접속한 TCP/IP 네트워크 연결 표시
netstat -a : 모든 포트표시, 컴퓨터 이름으로 표시
netstat -n : 접속된 사용자 IP주소로 표시
netstat -an : 모든 접속 내용을 IP로 표시
nbtstat -a [컴퓨터이름] : 컴퓨터이름으로 정보 확인
nbtstat -A [IP 주소] : IP주소로 정보 확인
net share : 현재 공유된 자원확인
net view : 네트워크 접속자 확인
net [start/stop] [실행/중지하려는 서비스명] : 윈도우내 프로그램의 실행과 중지
arp -a : IP 주소와 매칭되는 MAC 주소 표시
hostname : 현재 호스트의 이름 표시
nslookup [도메인] : DNS서버에 도메인의 IP 주소 찾기
